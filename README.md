# pocket

A dead-simple file clipboard for your terminal.

```bash
# Save file refs to your pocket
pocket foo.txt bar.pdf docs/

# Copy them to the current directory
pocket -r

# Move them instead
pocket -r -c

# See what's inside
pocket -l

# Remove item #2 from the clipboard
pocket -d 2
```
